# SEO 기초 가이드 (구글 공식 권장사항 기반)

## SEO란?
SEO(Search Engine Optimization, 검색 엔진 최적화)는 웹사이트가 구글 등 검색 엔진의 자연 검색 결과에서 더 높은 순위에 노출되도록 최적화하는 과정입니다.

## 구글이 중요시하는 핵심 요소

### 1. 콘텐츠 품질 (E-E-A-T)
- Experience(경험): 실제 경험에 기반한 콘텐츠
- Expertise(전문성): 주제에 대한 전문 지식
- Authoritativeness(권위성): 해당 분야에서의 신뢰도
- Trustworthiness(신뢰성): 정확하고 정직한 정보

### 2. 기술적 SEO
- 사이트 크롤링 가능성 (robots.txt, sitemap.xml)
- 모바일 친화성 (모바일 퍼스트 인덱싱)
- 페이지 로딩 속도 (코어 웹 바이탈)
- HTTPS 보안
- 구조화된 데이터 마크업

### 3. 온페이지 SEO
- 타이틀 태그 최적화
- 메타 디스크립션 작성
- 헤딩 태그(H1~H6) 구조화
- 이미지 alt 텍스트
- 내부 링크 구조
- URL 구조 최적화

### 4. 오프페이지 SEO
- 양질의 백링크 확보
- 소셜 시그널
- 브랜드 멘션
- 로컬 SEO (Google Business Profile)

## 피해야 할 SEO 관행 (블랙햇)
- 키워드 스터핑 (과도한 키워드 반복)
- 숨김 텍스트/링크
- 클로킹 (검색엔진과 사용자에게 다른 콘텐츠 표시)
- 저품질 백링크 구매
- 자동 생성 콘텐츠 (스팸성)
- 도어웨이 페이지

## 최신 트렌드 (2024-2025)
- AI 생성 콘텐츠에 대한 구글의 입장: 품질이 중요하지, 생성 방법은 무관
- 검색 생성 경험(SGE/AI Overview)의 등장
- 사용자 경험(UX) 시그널의 중요성 증가
- 비디오 SEO의 성장
